# 🏦 Bank Management System

A production-style desktop banking application built with C# and SQL Server, focused on clean architecture, reliability, and real-world banking workflows.

---

## 📌 Overview

Bank Management System is a structured desktop application designed to manage core banking operations such as client accounts, financial transactions, and authentication.

The project reflects practical software engineering practices, including separation of concerns, database normalization, and maintainable code structure.

---

## ✨ Features

* Secure authentication system (PIN-based)
* Full client management (Create, Update, Delete)
* Deposit and withdrawal operations
* Transaction history tracking
* Support for multiple transaction types
* Structured and normalized SQL database
* Responsive and organized user interface

---

## 🏗️ Architecture

* Layered structure separating UI and data access
* ADO.NET for database interaction
* Modular and maintainable codebase
* Designed for scalability and future extension

---

## 🛠️ Tech Stack

* C# (.NET Framework)
* Windows Forms (WinForms)
* SQL Server
* ADO.NET

---

## 🗄️ Database Design

The system is built on a relational database with well-structured tables:

* Clients
* Transactions
* TransactionTypes
* Users
* LoginRegisters
* Currencies

---

## ⚙️ Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/BankManagementSystem.git
   ```

2. Open the solution in Visual Studio

3. Restore the database:

   * Import the provided `.bak` file
   * Or execute SQL scripts

4. Configure the connection string

5. Run the application
